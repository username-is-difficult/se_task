##背景介绍
本次的程序为软件工程的课后习题的小demo

<a name="项目说明"></a>
##项目说明
本项目采用java语言进行开发。程序包含一个Hw类和一个Jrl监听类用于实现对于JradioButton的监听。
Hw类为窗口类，包含程序所有的面板、文本框、按钮等诸多控件，用于完成GUI图形界面（但java语言实现的GUI用户图形界面确实效果不理想）
项目的功能主要在actionPerformed里面实现，完成了对于习题的计算

<a name="使用说明"></a>
##使用说明
1.程序中包含年份、计算机字长、存储容量、价格、成本
2.输入要年份和选择好相应的计算机字长，点击确定
3.存储容量、价格、成本结果随后显示
